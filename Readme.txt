Requirement

How to Setup ? 
1. Install Python Version 2.7, Please following the official website 
>>> https://www.python.org/download/releases/2.7/

2. Install Robot Framework and Install the required packages followings.
- pip install robotframework
- pip install Selenium2Library
- pip install robotframework-jsonlibrary
- pip install robotframework-httplibrary

3. Setup WebDriver.
- go to https://chromedriver.chromium.org/downloads, download the latest version and ensure its the same with your chrome version.
- go to python directory (Python27), and create new folder as name "webdriver",  then bring the webdriver here.

3. Install IDE tools that you are comfortable with. eg. PyCharm, Sublime, RED, Eclipse IDE (Optional, if you want to edit code) 


How to Run ?
- Easy !, just cd to the directory of testcase, then run command.

>>>  yourdirectory/Zipmex Regression/testcase/web_ui
>>>  robot 01__get_best_prices.robot
